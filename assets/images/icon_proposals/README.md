## Choisir une icône pour l’app

1. Renomme la PNG choisie en `app_icon.png` (ou garde un nom clair).
2. Remplace `assets/images/app_icon.png` par ce fichier (ou mets le chemin dans `pubspec.yaml` sous `flutter_launcher_icons` → `image_path`).
3. Lance : `dart run flutter_launcher_icons`

Pour le **store**, prévoir une version **1024×1024 px** sans transparence si exigé par Apple/Google.

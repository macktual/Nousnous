# Texte à envoyer avec l’APK (WhatsApp)

Copier-coller le bloc ci-dessous dans votre message. Adapter le nom du fichier si besoin.

---

**Bonjour,**

Voici l’application **easynounou** (gestion assistante maternelle) au format **APK** pour **Android**.

### Installation sur votre smartphone

1. **Téléchargez** le fichier APK depuis WhatsApp (touchez le fichier puis **Télécharger** ou **Enregistrer**).
2. Ouvrez le **gestionnaire de fichiers** ou les **Téléchargements** et touchez le fichier **`.apk`** pour lancer l’installation.
3. Si le téléphone demande une autorisation :
   - Allez dans **Paramètres** → **Applications** → **Accès spécial** (ou **Installer des applications inconnues**),
   - Choisissez **Fichiers**, **Chrome** ou **WhatsApp** (selon ce que vous utilisez pour ouvrir l’APK),
   - Activez **Autoriser à partir de cette source**.
4. Relancez l’installation du fichier APK et appuyez sur **Installer**.

**Note :** selon la marque (Samsung, Xiaomi, etc.), les libellés peuvent varier un peu, mais l’idée est toujours d’**autoriser l’installation d’applications en dehors du Play Store** pour cette source une fois.

### Mise à jour

Si vous aviez déjà une version installée, installez simplement le **nouvel APK** par-dessus : vos données sont en principe **conservées**.

### En cas de problème

Écrivez-moi en précisant la **marque** du téléphone et le **message d’erreur** affiché.

Bonne utilisation.

# easynounou — Notice de démarrage (bref mode d’emploi)

---

## 1. Au premier lancement

1. Ouvre l’app : tu arrives sur **Mes contrats** (liste des enfants accueillis).
2. Touche l’icône **personne** en haut à droite → **Profil assistant maternel**.
3. Renseigne au minimum : **identité**, **adresse**, **agrément** (numéro, date, nombre d’enfants), **téléphone / e-mail** si tu veux qu’ils apparaissent sur les PDF.  
4. Enregistre avec **Enregistrer** (en haut à droite sur certains écrans).

Sans profil complet, certains **PDF** (ex. déclaration arrivée / départ) seront moins remplis.

---

## 2. Ajouter un enfant

1. Sur **Mes contrats**, touche **Ajouter un enfant** (bouton en bas à droite).
2. Saisis les infos : nom, prénom, date de naissance, parents, dates de contrat, etc.
3. **Enregistre** : l’enfant apparaît dans la liste **Enfants accueillis**.

---

## 3. Fiche d’un enfant

Touche une ligne dans la liste pour ouvrir la **fiche enfant**. Tu y trouves notamment :

- **Modifier** : profil, parents, horaires, photo, etc.
- Accès aux modules : **Vaccinations**, **Médicaments**, **Maladies**, **Doliprane**, **PDF** (fiche enfant, déclaration, contrat…), **Archives** (quand l’accueil se termine).

Les **horaires** peuvent comporter **plusieurs semaines types** ; les **changements** d’horaires se font avec une **date d’effet** (bouton du type « Modifier les horaires… »).

---

## 4. Écrans utiles sur l’accueil

| Icône en haut | Rôle |
|---------------|------|
| **i** (info) | **À propos** : version de l’app, mentions, don PayPal, sauvegarde iCloud (iPhone selon configuration). |
| **Archives** | Enfants dont l’accueil est **clos** et documents associés. |
| **Personne** | **Profil assistant** à mettre à jour si tes infos changent. |

---

## 5. Photos, caméra, notifications

- **Photos / caméra** : la première fois, accepte les **autorisations** demandées par le téléphone (galerie, appareil photo).
- **Doliprane (rappels)** : sur Android / iOS, accepte les **notifications** si tu veux les rappels d’ordonnance.

---

## 6. Données et confidentialité

Tout est stocké **sur ton appareil**. Pas d’envoi des données sur un serveur par l’app — voir le texte de **présentation** / **politique de confidentialité** si tu en distribues une.

---

## Version très courte — WhatsApp (résumé)

1. Ouvre **easynounou** → icône **personne** : remplis ton **profil assistant** et enregistre.  
2. **Ajouter un enfant** → remplis la fiche → ouvre l’enfant pour **vaccins**, **PDF**, **Doliprane**, etc.  
3. Icônes en haut : **À propos**, **Archives**, **Profil**.  
4. Autorise **photos** et **notifications** si le téléphone le demande.

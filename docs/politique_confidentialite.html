Politique de confidentialité — NousNous

Application : NousNous (Assistante maternelle)
Dernière mise à jour : 2026-03-25
Éditeur : Franck Tual

1. Objet

Cette politique décrit comment l’application NousNous traite les données à caractère personnel, dans le respect du RGPD et de la loi « Informatique et Libertés ».

2. Responsable du traitement

Contact : tualfranck@gmail.com
Adresse : France

3. Données traitées

Les données suivantes peuvent être enregistrées uniquement sur votre appareil :

  • Profil professionnel : civilité, nom, prénom, adresse, téléphone, e-mail, agrément (date, nombre d’enfants), code d’accès, étage.
  • Enfants : nom, prénom, date de naissance, photo, dates de contrat, particularités d’accueil.
  • Parents : nom, prénom, adresse, code postal, ville, téléphone, e-mail.
  • Santé / suivi : vaccinations (dates, justificatifs, photo), médicaments, maladies — dans le cadre strict de votre activité professionnelle.

Les données sont stockées sur l’appareil. Toutefois, l’utilisateur peut activer une sauvegarde iCloud optionnelle : lorsque la fonction « Enregistrer sur iCloud » est utilisée, une copie de sauvegarde (base + photos) est copiée et stockée dans le conteneur iCloud Drive de l’application (services fournis par Apple).

4. Finalités et base légale (RGPD)

FinalitéBase légale
Gestion des fiches, contrats, vaccins, médicaments, documents PDFExécution du contrat / intérêt légitime (activité professionnelle)

5. Destinataires

Aucune donnée n’est transmise à des serveurs de l’éditeur ni à des tiers à des fins de publicité/analytics. Les données restent sur votre téléphone ou tablette. En cas d’utilisation de la sauvegarde iCloud, la sauvegarde est stockée dans iCloud Drive (infrastructure d’Apple), sur le même compte Apple.

6. Durée de conservation

Les données sont conservées tant que l’application est installée. En cas de désinstallation, les données locales sont supprimées avec l’application. L’éditeur ne conserve aucune copie.

7. Vos droits (RGPD)

Vous disposez des droits d’accès, de rectification, d’effacement, de limitation, de portabilité et d’opposition. Pour les données stockées sur l’appareil, la désinstallation supprime les données. Pour exercer vos droits : tualfranck@gmail.com. Réclamation possible auprès de la CNIL.

8. Sécurité

Les données sont stockées dans l’espace réservé à l’application (sandbox). Il vous appartient de sécuriser l’accès à votre appareil.

En résumé : Les données restent sur votre appareil. Une sauvegarde iCloud est possible uniquement si vous l’activez dans l’app (stockage dans iCloud Drive). Vous pouvez supprimer les données en désinstallant l’application. Contact : tualfranck@gmail.com.

Toute modification de cette politique sera indiquée par une mise à jour de la date ci-dessus.

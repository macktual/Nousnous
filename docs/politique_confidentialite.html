3. Données traitées

Les données suivantes peuvent être enregistrées uniquement sur votre appareil :

  • Profil professionnel : civilité, nom, prénom, adresse, téléphone, e-mail, agrément (date, nombre d’enfants), code d’accès, étage.
  • Enfants : nom, prénom, date de naissance, photo, dates de contrat, particularités d’accueil.
  • Parents : nom, prénom, adresse, code postal, ville, téléphone, e-mail.
  • Santé / suivi : vaccinations (dates, justificatifs, photo), médicaments, maladies — dans le cadre strict de votre activité professionnelle.

Aucune de ces données n’est envoyée sur un serveur.

5. Destinataires

Aucune donnée n’est transmise à des serveurs ni à des tiers. Les données restent sur votre téléphone ou tablette. Les PDF que vous générez sont sous votre seule responsabilité.

6. Durée de conservation

Les données sont conservées tant que l’application est installée. En cas de désinstallation, les données locales sont supprimées avec l’application. L’éditeur ne conserve aucune copie.

7. Vos droits (RGPD)

Vous disposez des droits d’accès, de rectification, d’effacement, de limitation, de portabilité et d’opposition. Pour les données stockées sur l’appareil, la désinstallation supprime les données. Pour exercer vos droits : [VOTRE_EMAIL]. Réclamation possible auprès de la CNIL.

8. Sécurité

Les données sont stockées dans l’espace réservé à l’application (sandbox). Il vous appartient de sécuriser l’accès à votre appareil.

En résumé : NousNous ne transmet aucune donnée à un serveur. Tout reste sur votre appareil. Vous pouvez supprimer les données en désinstallant l’application. Contact : [VOTRE_EMAIL].
